Strategizer

Personal Project

Luau

Work-in-Progress

Strategizer start screen. Has 3 player icons for each team and a ball in the center

Strategizer Features:

  • Draggable objects
  • Drawing & Erasing
  • Working On Screen Buttons

Tentative Features:

  • Mimic/Copy a game inside a replay
  • Create and Save Powerpoints/Multiple Slides
  • Write on screen
  • Create strategies in 3D with 3D models

The Strategizer is an unreleased spectator script for Orion Drift that utilizes the drawing ability of the API to create it's own GUI on the screen. More specifically it creates multiple draggable objects on screen that represent players and the game ball. It also allows the user to draw on screen with a vertity of colors that are selected using custom button designs.
